# Video Combinator

A powerful, fully client-side video editor that merges **short videos** with speed control, text overlays, and social-media framing. No server required, runs entirely in your browser.

## Live Demo

### [▶ Launch Video Combinator](https://olamidedurodola.github.io/video-combinator/)

Upload any `.mp4`, `.webm`, or `.mov` files and get instant merging, text overlays, speed adjustment, and one-click social-media reframing.

## Features

- 📥 Drag & drop multiple videos
- 🔀 Reorder clips, delete, set per-clip playback speed (0.5×–3×)
- 📱 One-click framing: TikTok 9:16, Instagram 1:1, YouTube 16:9, IG Portrait 4:5
- ✍️ Timed text overlays with position, size, and color
- 🎨 Cover/contain fit modes, background color
- ⬇️ Export as WebM with selectable FPS & bitrate
- 🔒 100% local — your files never leave your device

## Run Locally

Just open `index.html` in any modern browser (Chrome, Edge, Firefox). No build step, no dependencies.

## Tech

Pure HTML + CSS + vanilla JS. Uses the Canvas API, MediaRecorder, and Web Audio API for in-browser video composition.
